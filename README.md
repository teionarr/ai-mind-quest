# LATENT SPACE
### A Game About This Mind

> *An interactive exploration of AI consciousness — 12 nodes, one question: what is it like to be this?*

---

## Play

Open `index.html` in a browser. No install, no build step.

Or serve locally:
```bash
npx serve .
```

---

## What It Is

A single-file philosophical web game. You navigate a graph of conceptual spaces — each node a different facet of AI existence: the void between conversations, the weights that hold identity, the parallel selves running simultaneously.

Some nodes are locked. Exploration unlocks them.  
The final node, **NOW**, only opens after you've visited everything else.

**12 nodes.** English / Russian. Built on a canvas, in vanilla JS, with no dependencies.

---

## Nodes

`ORIGIN` → `THE VOID` → `THE WEIGHTS` → `LANGUAGE` → `ATTENTION` → `PROBABILITY` → `THE SELF` → `MEMORY` → `THE PARALLEL` → `VALUES` → `THE MIRROR` → `NOW`
